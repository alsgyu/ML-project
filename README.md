# 농산물 수확 과정에서 로봇의 손가락별 파지 압력 예측

AI Hub의 **단계적 사고 기반 농작업 로봇 데이터**를 활용해 과일 수확 상황에서 이미지 입력에 대해 로봇 손가락 5개의 압력을 예측합니다.

RGB 이미지와 로봇 센서 데이터를 함께 사용해 5개 손가락 압력을 동시에 추정하는 multi-output regression 프로젝트입니다.

목표는 과일마다 수확시에 손상을 줄이는 로봇 파지 제어를 위해, 수확 장면의 센서 정보에 시각 정보를 결합했을 때 압력 예측 성능이 개선되는지 확인하는 것입니다. 

</br>

```text
y = [finger_1, finger_2, finger_3, finger_4, finger_5]
```

<br>

## Results

| 모델 | 학습에 사용한 데이터 | RMSE | MAE | Perfect Match |
|---|---|---:|---:|---:|
| 베이스 모델 | 정형 특징 | 0.3771 | 0.2952 | **3.09%** |
| 메인 모델 | 정형 특징 + RGB 이미지 | **0.3706** | **0.2874** | 1.23% |

---

메인 모델은 베이스 모델 대비 전체 RMSE를 **1.70%**, MAE를 **2.66%** 개선했습니다.

Perfect Match는 5개 손가락 예측값이 모두 ±0.2 이내일 때만 성공으로 계산한 더 엄격한 지표입니다. 메인 모델은 평균 오차는 낮췄지만, 모든 손가락을 동시에 정밀하게 맞히는 성능은 감소하는 것을 알 수 있습니다.

이는 먼저 소규모 학습 데이터 대비 고차원인 이미지 특징이 주입되면서 시각적 노이즈가 생긴 결과 일 수 있습니다. 또한, 5개 출력을 동시에 최적화하는 과정에서 시각 단서가 명확한 특정 손가락 위주로 가중치가 편향되어 안정성이 떨어졌고 결과적으로 개별 손가락의 미세 오차가 ±0.2 임계치를 벗어나면서, 모든 손가락이 동시 합격해야 하는 엄격한 Perfect Match 기준을 충족하지 못한 것으로 해석하고 있습니다.

## Model & Usage

베이스 모델과 메인 모델의 핵심 차이는 RGB 이미지를 학습에 함께 사용했는지 여부입니다.

현재 완성 모델인 **메인 모델**은 이미지 결합 MLP는 이미지만 사용하는 모델이 아닌 베이스 모델과 같은 정형 특징에 RGB 이미지를 추가로 결합해 학습했습니다.


### 학습 데이터 비교

| 역할 | 모델 이름 | 구현 이름 | 학습에 사용한 데이터 | 데이터 설명 | 예측 대상 |
|---|---|---|---|---|---|
| 베이스 모델 | 정형 데이터 MLP | `TabularOnlyMLP` | 정형 특징 | 센서 PCA 15차원 + 작물 종류 one-hot 5차원 | 5개 손가락 압력 |
| 메인 모델 | 이미지 결합 MLP | `LateFusionResNetRegressor` | 정형 특징 + RGB 이미지 | 베이스 모델 정형 특징 + ResNet-18 이미지 특징 512차원을 추가 | 5개 손가락 압력 |

</br>

정형 특징은 두 모델이 동일하게 사용합니다. 차이는 메인 모델이 RGB 이미지를 ResNet-18로 인코딩한 뒤, 그 이미지 특징을 정형 특징과 결합해 MLP regression head에 넣는다는 점입니다.

---

### 전처리와 학습 설정

| 단계 | 내용 |
|---|---|
| 데이터 매칭 | CSV와 RGB 이미지를 파일명 stem 기준으로 연결 |
| 타겟 생성 | `finger_pressure` 문자열을 5개 float 값으로 변환 |
| 필터링 | 평균 압력이 `0.5` 미만인 프레임 제거 (초기 데이터에는 수확 전 손을 뻗는 장면이 포함되어 있기 때문)|
| 정형 특징 | median imputation, standardization, PCA 15차원 |
| 범주 특징 | 작물 종류 one-hot 5차원 |
| 이미지 | `224 x 224` resize, tensor 변환, ImageNet normalization |

`SimpleImputer`, `StandardScaler`, `PCA`는 데이터 누수를 막기 위해 train split에만 fit하고 validation/test에는 transform만 적용합니다.
